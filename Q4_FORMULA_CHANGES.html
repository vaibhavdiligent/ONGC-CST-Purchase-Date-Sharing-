Formula Changes: Original vs Amended Q4 Discount

Original: YRVG004_QAIS_EXECUTE  |  Amended: YRVG004_QAIS_Q4_AMEND

Original Code (Removed / Changed) Amended Code (New / Added)

Change 1 — BCQ / QCQ Base Definition

Aspect Original Program Amended Program
What is BCQ? Full quarter committed qty (all 3 months)
QCQ = Jan MCQ + Feb MCQ + Mar MCQ
w_quater_min = (commited_qty_m10 + commited_qty_m11 + commited_qty_m12) * min_perc_q4 / 100
 Bi-monthly committed qty (Jan+Feb only)
BCQ = Jan MCQ + Feb MCQ
lv_bcq = commited_qty_m10 + commited_qty_m11 lv_qcq = commited_qty_m10 + commited_qty_m11 + commited_qty_m12

Change 2 — Slab % Calculation Formula

Aspect Original Program Amended Program
Slab % Formula Single calculation using full quarter lifting vs full QCQ
( grp_lift_qty_m10 + grp_lift_qty_m11 + grp_lift_qty_m12 ) / ( commited_qty_m10 + commited_qty_m11 + commited_qty_m12 ) * 100
Used implicitly via min/max threshold comparisons only. 		Two separate calculations:
-- Amended %: Jan+Feb lift vs BCQ lv_amended_pct = ( grp_lift_qty_m10 + grp_lift_qty_m11 ) / lv_bcq * 100 -- Original %: Full quarter vs QCQ lv_orig_pct = ( grp_lift_qty_m10 + grp_lift_qty_m11 + grp_lift_qty_m12 ) / lv_qcq * 100

Change 3 — Slab Thresholds

Aspect Original Program Amended Program
Min Threshold 85% of QCQ (from yrva_mstr_waiver-min_perc_q4)
w_quater_min = QCQ * 85 / 100
Below 85% → not eligible 		100% of BCQ (fixed)
IF p_perc GE 100 → eligible
Below 100% → NA (not eligible)
Slab Breakpoints Max cap: 125% of QCQ (from yrva_mstr_waiver-max_perc_q4)
No intermediate slabs — single rate from KONM scale 		Three fixed breakpoints:
IF p_perc GE 120 → p_rate = 2500 ELSEIF p_perc GE 110 → p_rate = 2000 ELSEIF p_perc GE 100 → p_rate = 1500 ELSE → p_rate = 0 (NA)

Change 4 — Discount Rate Source

Aspect Original Program Amended Program
Rate Source Dynamic lookup from SAP condition table
SELECT from A350 (kschl='ZQIS') → SELECT from KONM (scale table) SORT i_cond BY kstbm DESCENDING. LOOP AT i_cond WHERE kstbm LE mou_qty. w_kbetr = i_cond-kbetr. ENDLOOP.
 Fixed rates from new FORM GET_Q4_SLAB_RATE
FORM get_q4_slab_rate USING p_perc CHANGING p_rate. IF p_perc GE 120. p_rate = 2500. ELSEIF p_perc GE 110. p_rate = 2000. ELSEIF p_perc GE 100. p_rate = 1500. ENDIF. ENDFORM.
No A350 / KONM lookup for Q4.

Change 5 — Fallback / Comparison Logic

Aspect Original Program Amended Program
New Customer Handling Hard exclusion — record skipped entirely
IF mou_begda GE '20240201'. lv_flag123 = 'X'. EXIT. ENDIF.
Customer with MOU from Feb-2024 gets NO discount — no comparison done. 		Compare both schemes — give higher rate
PERFORM get_q4_slab_rate USING lv_amended_pct CHANGING lv_amended_rt. PERFORM get_q4_slab_rate USING lv_orig_pct CHANGING lv_orig_rt. IF lv_orig_rt GT lv_amended_rt. lv_q4_amended_rate = lv_orig_rt. scheme = 'ORIGINAL'. ELSEIF lv_amended_rt GT 0. lv_q4_amended_rate = lv_amended_rt. scheme = 'AMENDED'. ELSE. lv_flag123 = 'X'. "both NA scheme = 'NA'. ENDIF.

Change 6 — Discount Amount Calculation

Aspect Original Program Amended Program
Amount Formula Single combined amount: all 3 months together
tot_elgl_qty = ind_elgl_qty_m1 (Jan) + ind_elgl_qty_m2 (Feb) + ind_elgl_qty_m3 (Mar) value = tot_elgl_qty * w_kbetr
Jan, Feb, Mar not shown separately. 		Split: Jan+Feb amount and Mar amount shown separately
jan_feb_amt = ( ind_elgl_qty_m1 (Jan) + ind_elgl_qty_m2 ) (Feb) * lv_q4_amended_rate mar_amt = ind_elgl_qty_m3 (Mar) * lv_q4_amended_rate value = jan_feb_amt + mar_amt

Change 8 — New FORM Added

Aspect Original Program Amended Program
New Form Not present
Rate looked up from KONM for all quarters. 		FORM GET_Q4_SLAB_RATE added
FORM get_q4_slab_rate USING p_perc TYPE p CHANGING p_rate TYPE kbetr. CLEAR p_rate. IF p_perc GE 120. p_rate = 2500. ELSEIF p_perc GE 110. p_rate = 2000. ELSEIF p_perc GE 100. p_rate = 1500. ENDIF. ENDFORM.
Called twice per customer: once for amended%, once for original%.

Summary of All Changes

No. What Changed Original Amended
1 BCQ Definition Jan + Feb + Mar MCQ (full quarter) Jan + Feb MCQ only (bi-monthly)
2 Slab % Formula Full quarter lift / Full QCQ × 100 Two formulas: Amended (Jan+Feb / BCQ) AND Original (Full / QCQ)
3 Min Threshold 85% of QCQ (from master waiver table) 100% of BCQ (fixed)
4 Slab Breakpoints 125% cap — single rate from KONM scale 100% → 1500 / 110% → 2000 / 120% → 2500 (fixed)
5 Rate Source Dynamic from A350 → KONM condition table Fixed slab amounts via GET_Q4_SLAB_RATE
6 Fallback Logic MOU from Feb-2024 → skip entirely (EXIT) Compare both schemes → apply higher rate
7 Amount Split One combined value for Jan+Feb+Mar Jan+Feb amount and Mar amount shown separately
8 Output Columns Tot Grp Lift Qty, Total Eligible Qty, Discount + Amended BCQ%, Original QCQ%, Scheme, Jan+Feb Amt, Mar Amt
9 New FORM Not present GET_Q4_SLAB_RATE added

Note: All changes are isolated to Q4 (January–March) processing only. Q1 (Apr–Jun), Q2 (Jul–Sep), Q3 (Oct–Dec) and all other discount types (Annual, Monthly, Consistency) are unchanged.
